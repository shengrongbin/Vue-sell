## 1、根据经纬度获取位置详情

### 请求URL：
	http://localhost:4000/position/:geohash

### 示例：
[http://localhost:4000/position/40.10038,116.36867](http://localhost:4000/position/40.10038,116.36867)

### 请求方式：
	GET

### 参数类型：param

	|参数	  |是否必选 |类型     |说明
	|geohash    |Y       |string   |经纬度

## 6、用户名密码登陆

### 请求URL：
	http://localhost:4000/login_pwd

### 请求方式：
	POST

### 参数类型: 请求体

	|参数	  |是否必选 |类型     |说明
	|name       |Y       |string   |用户名
	|pwd        |Y       |string   |密码
	|captcha    |Y       |string   |验证码

### 返回示例：
    * 登陆成功
      {
        "code": 0,
        "data": {
          "_id": "5a9cd9c6ad5b2d34d42b385d",
          "name": "aaa"
        }
      }
    * 登陆失败
      {
        "code": 1,
        "msg": "用户名或密码不正确!"
      }
